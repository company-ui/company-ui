# Company-UI

Works with both Vue 2 , Vue 3 And React.

# Translations

- [中文文档](README_zh.md)

## Installing

```bash
# or pnpm or yarn
npm install @company-ui/dom
```

## Usage

### General

```ts
import { CompanyAutocomplete } from '@company-ui/dom';

const companyAutocomplete = new CompanyAutocomplete({
  target: '#example',
});
```
