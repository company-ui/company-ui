# `@company-ui/dom`

Works with both Vue 2 , Vue 3 And React.

## Installing

```bash
# or pnpm or yarn
npm install @company-ui/dom
npm install @company-ui/style
```

## Usage

### General

```ts
import '@company-ui/style';
import { Autocomplete } from '@company-ui/dom';

new Autocomplete({
  target: '#example',
});
```
