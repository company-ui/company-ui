# `@company-ui/autocomplete-core`

It exposes APIs to `@company-ui/autocomplete-dom`, `@company-ui/vue`, and `@company-ui/autocomplete-react`.

## Installing

```bash
# or pnpm or yarn
npm install @company-ui/autocomplete-core
```
